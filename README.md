## 🛠 Troubleshooting Common Issues

- **The app won't start or crashes:** Make sure your Windows is updated and meets system requirements. Try restarting your PC and reinstalling the app.  
- **Internet connection problems:** The app needs internet access to get live data. Check that your connection is stable. If you use a firewall or antivirus, allow polymarket-terminal through.  
- **Market data not loading:** Refresh the app by closing and reopening it. Also, verify that Polymarket is online and functional.  
- **Installation blocked by Windows Defender:** When Windows blocks the installer, click “More info” then “Run anyway” to proceed.
